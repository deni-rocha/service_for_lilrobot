Edit Commands

Cursos Movement Commands

  1. Home - Move to the start of the current line
  2. End - Move to the end of the current line
  3. Ctrl+Up - Scroll up one line
  4. Ctrl+Down - Scroll down one line
  5. PageUp - Scroll up one screen
  6. PageDown - Scroll down one screen
  7. Crtl + PgUp - Scroll left one screen
  8. Crtl + PgDn - Scroll right one screen
  9. Crtl + Home - Scroll to the of the document
  10. Crtl+End - Scroll to the endd of the document
  11. Crtl+Left - Move left one word
  12. Crtl+Right - Move right one word
  13. Ctrl+Up - Scroll up one line
  14. Ctrl+Down - Scroll down one line
  15. PageUp - Scroll up one screen
  16. PageDown - Scroll down one screen
